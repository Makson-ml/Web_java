BoxingCost — показывает, почему int быстрее Integer. При работе с Integer Java тратит время на создание объектов и проверку на null. Три метода: сложение с boxing, без boxing, и конвертация между ними.

User — простой класс пользователя с полями id, email и age. В конструкторе проверяем, что email не пустой, а возраст от 0 до 150. Если данные неверные — выбрасываем своё исключение InvalidFieldException. Методы equals, hashCode и toString работают по всем полям.

FunInfinity — бесконечный цикл через переполнение числа. Когда int достигает максимума и увеличивается ещё на 1, он становится отрицательным. Условие цикла остаётся истинным, и программа зависает навсегда

Homework — считает интеграл методом прямоугольников. Разбиваем отрезок на кусочки, считаем площадь каждого и складываем. Проверяем, что шаг положительный и начало отрезка не больше конца.
